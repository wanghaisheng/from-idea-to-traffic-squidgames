# i18n Translation Files

    This document outlines the structure and content of our translation files.

    ## File Structure

    Translation files will be stored in a dedicated directory, for example, `i18n/`. Each language will have its own JSON file.

    ```
    i18n/
    ├── en.json
    ├── es.json
    └── fr.json
    ```
